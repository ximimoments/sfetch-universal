# sfetch
A simple system fetch CLI tool written in Shell. (For another distro)

## Installation
Run `sudo make install` to install the system fetch tool. If you're on  ~~Arch Linux~~ Other Distro , there's an ~~AUR~~ Official Repo package Universal for [sfetch](https://aur.archlinux.org/packages/sfetch). You can install it with your favorite AUR helper, in my case its paru: `paru -S sfetch`.
